# Sejam bem-vindos!

Este é o meu blog pessoal, criado com o objetivo de compartilhar ideias, experiências e aprendizados nas áreas de desenvolvimento de software e tecnologia.

## Sobre o projeto

Desenvolvido com [Astro](https://astro.build/) - e o tema [Astro Resume Theme](https://github.com/wasutz/astro-resume-theme) - um framework web javascript moderno focado na criação de sites orientados a conteúdo. O Astro permite gerar HTML estático por padrão e só carrega JavaScript quando necessário - graças a arquitetura Astro Islands (ilhas) - garantindo excelente desempenho e melhorando as taxas de conversão.

## Por que o Astro?

1. Uma arquitetura web baseada em componentes otimizada para sites orientados a conteúdo;
2. Suporte a diversos frameworks de UI (React, Vue, Svelte, Solid, etc);
3. Estilização com Tailwind CSS, facilitando o design responsivo e utilitário;
4. Sistema de roteamento automático baseado em arquivos;
5. Integração fácil com Markdown e MDX, ideal para blogs e documentação técnica;
6. Deploy automatizado via GitHub Actions para integração contínua.

## O que vem a seguir?

Este blog servirá como espaço para publicar artigos, tutoriais, experiências e insights sobre desenvolvimento web moderno, ferramentas open-source e boas práticas.

Obrigado por visitar!
